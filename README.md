# Spring Boot Learning Sandbox

This repository is dedicated to exploring and mastering Spring Boot fundamentals. It serves as a clean, practical space for building REST APIs and understanding core Java backend concepts.

### 🚀 Quick Start
To launch the application:
```bash
./mvnw spring-boot:run
```

### 🛠️ Tech Stack
- **Framework:** Spring Boot 3.x
- **Build Tool:** Maven
- **Java Version:** 17

### 📚 Core Focus
- REST API implementation
- Project configuration (`application.properties`)
- Spring project architecture
